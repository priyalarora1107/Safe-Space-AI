# 🛡️ Safe Space AI

Safe Space AI is a privacy-first AI journaling application that helps users reflect on their emotions, track their mental well-being, and gain meaningful insights—all while keeping their data stored locally on their own device.

## 🌟 Problem Statement

Many journaling and mental wellness applications rely on cloud storage, requiring users to upload deeply personal thoughts to external servers. This raises concerns about privacy, security, and ownership of sensitive personal data.

## 💡 Our Solution

Safe Space AI provides an AI-powered journaling experience where users can:

- Write daily journal entries
- Analyze emotional tone using AI
- View personalized emotional insights
- Track mood history over time
- Export their journals
- Keep all journal data stored locally for complete privacy

## ✨ Features

- 📝 Create and edit journal entries
- 🤖 AI-powered mood and sentiment analysis
- 📊 Emotional insights dashboard
- 📈 Mood history and trends
- 🌙 Dark mode support
- 💾 Local storage for complete privacy
- 📤 Export journal data
- 🗑️ Delete all stored data
- 📱 Responsive modern UI

## 🔒 Privacy First

Safe Space AI is designed with privacy as its core principle.

- No cloud storage
- No external database
- No user accounts
- No tracking
- All journal entries remain on the user's device

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Safe-Space-AI.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Open:

```
http://localhost:3001
```
